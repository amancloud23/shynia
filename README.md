# ✦ Shynia Beauty

A luxury cosmetics e-commerce website built with pure HTML, CSS, and JavaScript.

## Features

- 🌸 **Hero section** with animated blob and call-to-action buttons
- 🛍️ **Product catalogue** with filter by category (Skincare, Makeup, Fragrance, Hair Care)
- 🛒 **Shopping cart** sidebar with add/remove and quantity controls
- 💬 **Testimonials** from happy customers
- 📖 **About** section with brand story
- 📧 **Newsletter** signup and **Contact** form
- 📱 **Responsive** mobile-friendly layout
- ✨ **Scroll-reveal** animations

## Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (no dependencies)

## Running Locally

Just open `index.html` in any modern browser — no build step required.
